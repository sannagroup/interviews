# Sanna Interviews

Internal repository for take-home exercises used in Sanna's hiring process.

Each role has its own folder containing:

- `README.md` — candidate-facing brief
- `data/` — input data the candidate needs

## Available exercises

| Folder | Role | Time budget |
|---|---|---|
| [`ai-engineer/`](./ai-engineer) | AI Engineer | ~3 hours + 45–60 min walkthrough |

## How to give a take-home

1. Copy the role folder into a fresh private repo for the candidate, or zip and email it.
2. Share the `README.md` brief link.
3. Agree on a deadline (default: 5 calendar days).
4. After they submit, run the 45–60 min walkthrough as the primary signal for design and architecture.
